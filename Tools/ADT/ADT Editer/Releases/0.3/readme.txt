Tigurius' ADTEditer Release 0.3(alpha)
Compiled at Semptember 12th, 2008 (Bugs fixed and redesigned GUI ~still not as great as I expect gah)

Well, here my first ADT-Tool written in Delphi!

Functions available:

ShowTextureIDs:
Displays the TextureIDs

AddDiffGEID:
Adds GroundEffecID's to your adt's textures - starting of from the number given in the edit.text.box +4 

AddSameGEID:
Add one GEID to all textures in the adt

AddTexGEID:
Adds one GEID to the specified Texture

AddAreaID:
Adds an AreaID to the given Chunk
